We're using a mentorship platform called Minerva, and we're really excited to be piloting it!

How do I use Minerva?


Here's how it works: In any channel, simply use the /minerva help command. Remember to keep everything lowercase!
If you’d like to help other hackers, you can use the /minerva add command to tell Minerva the skills that you are familiar with.
Now, when a hacker asks for help with a skill that you know, you will be automatically matched with that hacker! When you’re done, use /minerva end to end your help session with a hacker.
If you’d like to see what you can do with Minerva, just type /minerva anywhere in the MHacks Slack.
